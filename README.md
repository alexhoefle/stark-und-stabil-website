# Stark und Stabil

Website für **Stark und Stabil** – Resilienz- und Lebenskompetenztraining für Kinder
von Karin Schubert.

Eine vollständig responsive, moderne Landingpage – gebaut mit reinem
**HTML5, CSS und JavaScript** (keine Frameworks).

## Struktur

| Datei | Inhalt |
|-------|--------|
| `index.html` | Semantisches HTML5-Markup aller Sektionen |
| `style.css` | Design-System (CSS-Variablen), Layout (Flexbox/Grid), Responsive, Animationen |
| `script.js` | Mobile-Navigation, Sticky-Header, Scroll-Reveal, Akkordeon, Scrollspy |

## Sektionen

1. Header mit Sticky-Navigation
2. Hero – „Gemeinsam für starke Kinder“
3. Warum Resilienztraining wichtig ist
4. Über mich – Karin Schubert
5. Pädagogisches Konzept (Akkordeon)
6. Angebote & flexible Formate
7. Zitat-Sektion
8. Kontakt
9. Footer

## Lokal öffnen

Einfach `index.html` im Browser öffnen – es werden keine Build-Tools benötigt.

## Hinweis zum Logo

Das Markenzeichen ist als Inline-SVG umgesetzt (Schild + Herz in den
Markenfarben). Liegt eine finale Logo-Datei vor, kann das `<svg class="logo__mark">`
in `index.html` und im Footer durch ein `<img>` ersetzt werden.
